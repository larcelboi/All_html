Exercice – Formulaire (validation par expressions régulières)

Vous devez modifier le code de la page index.html pour que les champs soient du bon type et validés selon les critères demandés.

Vous devez utiliser un mélange de validation HTML5 et d'expressions régulières.

Doit commencer par une lettre, 3 à 16 caractères (lettres, chiffres, soulignés).
8+ caractères dont minuscule, majuscule, chiffre et symbole.
nom@domaine
Format accepté : 123-456-7890, (123)456-7890, 123 456 7890
Format canadien : A1A 1A1 (espace ou sans).
Débute par le protocole, suivit de l'adresse.
Quatre octets 0–255 séparés par des points.
Rien à faire, un champ de type date se valide lui-même.
Valeur numérique entre 1 et 100.
